Плагин добавляет на сервер систему заработка через фермерство.

Игроки собирают пшеницу, которая автоматически отправляется в виртуальный рюкзак.
Когда рюкзак заполнен — его можно продать и получить деньги.
После сбора урожай восстанавливается через время, указанное в конфигурации.

-  Автоматический сбор урожая в рюкзак
-  Продажа собранных ресурсов за деньги
-  Система прокачки рюкзака (увеличение слотов)
-  Бусты (x2, x3 и т.д.):
-  Глобальные (на сервер)
-  Личные (на игрока)
-  Поддержка PlaceholderAPI
-  Отображение данных в TAB / scoreboard
-  Топ фермеров
